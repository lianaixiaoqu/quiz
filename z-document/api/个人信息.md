# 知识微答欢迎页面

## 当前文档版本: 1.1

### 说明：
1.个人活动个人信息总关数和累计已闯关关数显示

#### 关于个人活动统计数
--------------------------------
##### url: `/person/info`

请求方式 : `get`

参数名    | 含义    | 是否必须
-------|--------|-----
activityId  |活动 ID | Y


###  返回值

参数名  | 含义
-------------|-------------
sectionAmount     |已闯总关数
correctAmount|答对题目总数
correctRate       |正确率
beat    |打败多少人
Stars   |累积得到的星星数
